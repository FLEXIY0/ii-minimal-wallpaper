Minimal Wallpaper
=================

Установка:
curl -sL https://raw.githubusercontent.com/FLEXIY0/ii-minimal-wallpaper/main/random_minimal_wall.sh > minimal-wall && chmod +x minimal-wall && ./minimal-wall

Использование:
[→] - след.
[Enter] - применить
[s] - сохранить
[q] - выход

Совместимость:
- OS: Linux (любой дистрибутив)
- WM: Hyprland (нужен hyprpaper) ИЛИ тема Illogical Impulse
- Пакеты: curl, jq, chafa (для превью)
